# 🗺️ MacHub Development Roadmap

Tracking the evolution of MacHub from a seating tool to a full student ecosystem.

---

## ✅ Completed Features

### FEAT-0001: Unified Exam Hub Master view
- **Completed:** 18-05-2026 @ 11:32:04 PM
- **Goal:** Merge Timetable and Seating into one box.
- **Impact:** User stay time increased; navigation friction eliminated.

### FEAT-0002: Seamless Guest Mode (Skip Logic)
- **Completed:** 18-05-2026 @ 11:45:22 PM
- **Goal:** Allow instant access without setup.
- **Impact:** Removed barrier to entry for casual users.

### FEAT-0003: Modern Unified Data Format
- **Completed:** 19-05-2026 @ 01:29:51 AM
- **Goal:** Move from nested arrays `[[...]]` to rich objects `{hall: "", seat: ""}`.
- **Impact:** Allows for complex maps with multiple departments in one hall.

### FEAT-0004: PWA Home Screen Integration
- **Completed:** 18-05-2026 @ 10:33:49 PM
- **Goal:** Make the web app feel native.
- **Impact:** Added `apple-touch-icon`, `standalone` mode, and `black-translucent` status bar.
